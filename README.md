# Axiom Odyssey: New Epoch

A hardcore technology-focused Minecraft 1.7.10 modpack inspired by GregTech: New Horizons.

## Requirements

- Minecraft 1.7.10
- Forge
- Java 17+ (via [lwjgl3ify](https://github.com/GTNewHorizons/lwjgl3ify))

## Repository Structure

| Folder | Contents |
|--------|----------|
| `config/` | Mod configuration files |
| `scripts/` | MineTweaker 3 scripts (`.zs`) |
| `resourcepacks/` | Resource packs |
| `mods/` | Mod list / manifest |
| `journeymap/` | JourneyMap data |

## Development

Scripts support hot reload via `/mt reload` (MineTweaker 3).
